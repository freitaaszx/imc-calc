# 🧮 Calculadora de IMC em C#

Este é um projeto simples de uma calculadora de IMC (Índice de Massa Corporal) feita em C#, executada via terminal.

## 📌 Funcionalidades

* Entrada de altura e peso pelo usuário
* Cálculo automático do IMC
* Exibição do resultado
* Classificação básica do IMC

## 🧠 Fórmula utilizada

IMC = peso / (altura * altura)

## ▶️ Como executar

1. Compile o projeto em um ambiente C#
2. Execute o programa
3. Insira:

   * Sua altura (em metros, ex: 1.75)
   * Seu peso (em kg, ex: 70)

## 📊 Classificação de IMC

* Abaixo de 18.5 → Abaixo do peso
* 18.5 a 24.9 → Peso normal
* 25 a 29.9 → Sobrepeso
* 30 ou mais → Obesidade
